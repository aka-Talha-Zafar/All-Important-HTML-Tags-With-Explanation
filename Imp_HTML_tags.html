Complete HTML Tags Reference Guide

A comprehensive cheatsheet of essential HTML tags with examples and explanations for web developers

What is HTML:

HTML stands for HyperText Markup Language. It is the standard language used to "create and structure content on the web". It defines the layout and elements of a webpage using tags, such as headings, paragraphs, links, images, and more. It can be said the Skeleton of website.

Types of Tags in HTML

There are 2 types of tags in HTML

  1. Self-Closing Tag (Void Element)
  2. Paired Tag (Container Tag)

Self-Closing Tag (Void Element):

A self-closing tag does not need a closing tag because it doesn't wrap any content. It ends with /> (optional), It is used for empty elements.
Example: < br />, < image src=" " />, < hr >

Paired Tag (Container Tag):

A paired tag has an opening and a closing part and wraps around content to apply structure or styling.
Example: < p > Pargraph < /p >, < h1 > Heading < /h1 >

Attributes in HTML:

In HTML, attributes are special words used inside an opening HTML tag to provide additional information about an element. They modify the element's behavior, style, or functionality. Attributes are always specified within the opening tag of an element and usually come in name="value" / (Key = "value") pairs.
Example:
< img src="image.jpg" alt="Description" >               < !-- src and alt attributes -- >
< input type="text" placeholder="Enter name" >      < !-- type and placeholder -- >
< p id="intro" class="highlight">Hello < /p >           < !-- id and class -- >

Head Tag:

The < head > tag contains metadata and settings for the HTML document that are not directly visible on the page.It usually includes the Title, links to CSS files, scripts, fonts, and meta tags for SEO and responsiveness.
Example:
< head >
< title > My Website < /title >
< link rel="stylesheet" href="style.css" >
< /head >

Meta Tag:

The < meta > tag provides metadata about the HTML document, such as character encoding, author, keywords, site description, or viewport settings.It is placed inside the < head > tag and helps browsers and search engines understand the page better.
Example:
< meta charset="UTF-8" >
< meta name="viewport" content="width=device-width, initial-scale=1.0" >

Title Tag:

The < Title > tag is used to set the title of a web page. It is not only visible to users in the browser tab but is also used by browsers and search engines to understand the page content, which helps improve SEO.
Example: < title > "Your page title here" < title >

Link Tag:

The < link > tag is used to connect external resources to an HTML document, most commonly for linking CSS stylesheets. It is placed inside the < head > and does not display anything on the page itself.
Example: < link rel="stylesheet" href="style.css" >

Body Tag:

The < body > tag defines the main visible content of a web page, including text, images, buttons, and more. Everything inside the < body > is rendered by the browser for the user to see and interact with.
Example:
< body >
< h1 > Welcome < /h1 >
< p > This is my website.< /p >
< /body >

Heading Tag:

HTML provides six heading tags from < h1 > to < h6 >, used to define headings from most important (< h1 >) to least (< h6 >). They help structure content and improve accessibility and SEO.
Example:
< h1 > Main Heading < /h1 >
< h2 > Subheading < /h2 >

Paragraph Tag:

The < p > tag is used to define a paragraph of text in websites. Note: "It automatically adds spacing before and after the text to separate it from other content."
Example: < p > This is a paragraph. < /p >

Anchor Tag:

The "Anchor < a >" tag is used to create hyperlinks to other web pages, files, email addresses, or sections of the same page. It uses the "href" attribute to specify the destination URL.
Example: < a href="https://example.com" > Visit Website < /a >

Image Tag:

The < img > tag is used to display images on a web page.It is a self-closing tag and requires the src (image path) and alt (alternative text) attributes.
Example: < img    src="address of image"   alt="alternative text" >

Table Tag:

The < table > tag is used to create a table structure in HTML and acts as a container for all table-related elements like rows and cells.It organizes data into a grid of rows and columns.
Syntax: < table > ... < /table >

Table_Row Tag:

The < tr > tag defines a row in a table and is used to group a set of < th > or < td > cells horizontally.Every new < tr > adds a new horizontal row to the table.
Syntax: < tr > Table Row < tr >

Table_Heading Tag:

The < th > tag is used to define header cells in a table, which are usually placed in the first row.By default, the text in < th > is bold and centered, making it visually distinct.
Syntax: < th > Table Heading < th >

Table_Data Tag:

The < td > tag is used to define standard data cells in a table that contain the actual data.Each < td > goes inside a < tr > and represents a single cell in that row.

Basic Code Structure of a Table:

<table> 
  <tr>
    <th> Pet Name </th>
    <th> Superpower </th>
    <th> Favorite Snack </th>
  </tr>
  <tr>
    <td> Bob </td>
    <td> Laser Eyes </td>
    <td> Tuna Pizza </td>
  </tr>
  <tr>
    <td> Tom </td>
    <td> Mind Control </td>
    <td> Chicken Nuggets </td>
  </tr>
  <tr>
    <td> Simen </td>
    <td> Invisibility </td >
    <td> Popcorn </td>
  </tr>
</table>

Colspan Attribute:

The colspan is not an html tag, it's an attribute used in a < th > or < td > to make a single cell stretch across multiple columns.This is useful when you want one cell to occupy the space of two or more columns.
Syntax: < td colspan="2" > Merged Columns < /td >

Rowspan Attribute:

Similarly to colspan rowspan is also an attribute used in a < th > or < td > to make a single cell extend vertically (along X-axis) across multiple rows.
Syntax: < td rowspan="2" > Merged Rows < /td >

Ordered_List Tag:

The < ol > tag is used to create a numbered list, where each item appears in a specific sequence.Each item inside the list is wrapped in an < ol > tag.
Syntax:
< ol >
< li > ... < /li >
< li > ... < /li >
< /ol >

Unordered_List Tag:

The < ul > tag creates a bulleted list where the order of items is not important.Like < ol >, each item goes inside an < ol > tag.
Syntax:
< ul >
< li > ... < /li >
< li > ... < /li >
< /ul >

Defination_List Tag:

The < ul > tag is used for definitions, it pair terms with their descriptions.Inside it, < dt > defines the term, and < dd > gives the description.
Syntax:
< dl >
< dt > HTML < /dt >
< dd > HyperText Markup Language < /dd >
< /dl >

Form Tag:

The < form > tag is used to collect user input and send it to a server for processing (like login, signup, or search).It wraps around input fields, buttons, and other form elements, and uses action and method attributes to define where and how data is submitted.
Example: 

    <form action="/submit" method="post">
        <label for="name">Enter your name:</label>
        <input type="text" id="name" name="name">
        <label for="email">Email:</label>
        <input type="email" id="email" name="email">
        <button type="submit">Submit</button>
    </form>

Input Tag:

The < Input > tag is used to create interactive form controls like text fields, checkboxes, radio buttons, etc.It is a self-closing tag and requires the type attribute to define the kind of input.
Syntax: <input type="text" id="username" name="username">

Label Tag:

The < Label > tag is used to define a label for an input field, improving form accessibility and usability.It can be linked to an input using the for attribute, which matches the input's id.
Syntax: <label for="username">Username:</label>

Div Tag:

The < div > tag is a block-level container used to group/wrap HTML elements for styling or layout purposes.It doesn't add any visual change by default but is widely used with CSS or JavaScript to structure web pages.
Example:
<div class="container">
<p>Hello World!</p>
</div>

Textarea Tag:

The < textarea > tag is used to create a multi-line text input field where users can write longer content like comments or messages.It supports attributes like rows, cols, placeholder, and more, and requires both opening and closing tags.
Syntax: <textarea rows="4" cols="30" placeholder="Enter your message here"></textarea>

Select Tag:

The < select > tag is used to create a dropdown menu in a form, allowing users to choose one (or multiple) options.It works with nested < option > tags and supports attributes like name, id, and multiple.
Example:
<select name="fruits">
<option value="apple">Apple</option>
<option value="banana">Banana</option>
</select>

Span Tag:

The < span > tag is an inline container used to group/wrap a portion of text or elements for styling or scripting purposes. It doesn't affect layout by itself but is commonly used with CSS or JavaScript.
Example: <p>This is a <span style="color: red;">highlighted</span> word.</p>

Block Elements:

Block elements occupy the full width of their container and always start from a new line. They are mainly used for structuring the page layout and grouping content.
Example: <div>, <p>, <h1>, <ul>, <table>

Inline Elements:

Inline elements only take up as much width as necessary and do not break to a new line. They're used to style or manipulate small parts of text or content inside block elements.
Example: <span>, <a>, <strong>, <img>, <label>

ID Attribute:

The ID attribute uniquely identifies a single HTML element on a page. It must be unique and is often used for styling or JavaScript targeting.
Example: <div id="header">

Class Attribute:

The Class attribute assigns one or more labels to an element, allowing multiple elements to share the same styling or behavior.Also a single element can also have multiple classes allowing single element having multilpe stylings.
Example 1: (Single class to multiple Elements)
<p class="highlight">First paragraph.</p>
<div class="highlight">Some highlighted div content.</div>
Example 2: (Multiple classes to single element)
<p class="note important warning">This is a message.</p>

Audio Tag:

The < audio >> tag is used to embed sound content in a webpage, such as music or sound effects. It supports multiple file formats and provides controls like play, pause, and volume.
Example 1:
<audio controls>
<source src="song.mp3" type="audio/mpeg">
Your browser does not support the audio tag.
</audio>
Note: (This formate is used to enter multiple audio formates and the most compatible one with browser is selected automatically)
Example 2:
<audio src="test.mp3" controls></audio>
Note: (This formate is used when you only have one audio file and don't care about older browser support or format fallback.)

Video Tag:

The < video > tag is used to embed video content in a webpage. It supports controls for play, pause, fullscreen, etc., and allows multiple source formats for better compatibility.
Example 1:
<video width="320" height="240" controls>
<source src="video.mp4" type="video/mp4">
Your browser does not support the video tag.
</video>
Note: (This formate is used when you only have one video file and don't care about older browser support or format fallback.)
Example 2:
<video src="video.mp4" controls></video>

SVG Tag:

The < svg > tag defines a container for Scalable Vector Graphics (SVG), which are resolution-independent images created with XML syntax—ideal for logos, icons, charts, and illustrations.
Fun Fact: < img > tag is like saying: “Hey browser, just show this file.”But < svg > is like saying: “Hey browser, draw this yourself.”
Example:
<svg width="100" height="100">
<circle cx="50" cy="50" r="40" stroke="black" stroke-width="3" fill="red" />
</svg>

Iframe Tag:

The < iframe > (inline frame) tag is used to embed another HTML document (like a website, video, or map) inside the current HTML page.
Example:
<iframe src="https://www.example.com" width="600" height="400"> </iframe>

Header Tag:

The < header > tag represents the introductory content of a page or section — typically includes the site logo, title, or navigation links.It can appear multiple times in a document (e.g., one for the site, others for sections).
Example:
<header>
<h1>Welcome to My Website</h1>
</header>

Navigation (nav) Tag:

The < nav > tag is used to group navigation links like menus, sidebars, or internal anchors. It helps users move around the site and improves accessibility.
Example:
<nav>
<a href="#home">Home</a>
<a href="#about">About</a>
</nav>

Main Tag:

The < main > tag represents the primary content of a webpage, excluding sidebars, headers, or footers. Only one < main > element should exist per page.
Example:
<main>
<h2>Blog Post</h2>
<p>This is the main content area.</p>
</main>

Artical Tag:

The < artical > tag contains independent, self-contained content like a blog post, comment, or news item. Each article should make sense on its own.
Example:
<article>
<h2>Breaking News</h2>
<p>Some news content goes here.</p>
</article>

Aside Tag:

The < aside > tag defines secondary or side content, such as ads, tips, or sidebars. It's often used next to the main content but not part of it.
Example:

Figure Tag:

The < Figure > tag is used to group media elements (like images, charts, or diagrams) with optional captions using < figcaption >. It provides semantic meaning to visual content.
Example:
<figure>
<img src="image.jpg" alt="Example">
<figcaption>This is a caption.</figcaption>
</figure>

Section Tag:

The < section > tag is used to define a thematic group of related content, typically with a heading. It's useful for breaking content into logical blocks like services, chapters, or topics.
Example:
<section>
<h2>Our Services</h2>
<p>We offer web design and development.</p>
</section>

Footer Tag:

The < footer > tag defines the bottom section of a webpage or block, often containing copyright, links, or contact info.It can be used multiple times within a page.
Example:
<footer>
<p>© 2025 MyWebsite. All rights reserved.</p>
</footer>
