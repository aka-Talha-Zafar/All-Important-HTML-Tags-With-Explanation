All important HTML tags with there explanation


What is HTML:

HTML stands for HyperText Markup Language.It is the standard language used to "create and structure content on the web".It defines the layout and elements of a webpage using tags, such as headings, paragraphs, links, images, and more.It can be said teh Skeleton of website.

Types of Tags in HTML

There are 2 types of tags in HTML

  1. Self-Closing Tag (Void Element)
  2. Paired Tag (Container Tag)

Self-Closing Tag (Void Element):

A self-closing tag does not need a closing tag because it doesn't wrap any content. It ends with /> (optional), It is used for empty elements.
Example: < br />, < image src=" " />, < hr >

Paired Tag (Container Tag):

A paired tag has an opening and a closing part and wraps around content to apply structure or styling.
Example: < p > Pargraph < /p >, < h1 > Heading < /h1 >

Attributes in HTML:

In HTML, attributes are special words used inside an opening HTML tag to provide additional information about an element. They modify the element's behavior, style, or functionality. Attributes are always specified within the opening tag of an element and usually come in name="value" / (Key = "value") pairs.
Example:
< img src="image.jpg" alt="Description" >               < !-- src and alt attributes -- >
< input type="text" placeholder="Enter name" >      < !-- type and placeholder -- >
< p id="intro" class="highlight">Hello < /p >           < !-- id and class -- >

Head Tag:

The < head > tag contains metadata and settings for the HTML document that are not directly visible on the page.It usually includes the Title, links to CSS files, scripts, fonts, and meta tags for SEO and responsiveness.
Example:
< head >
< title > My Website < /title >
< link rel="stylesheet" href="style.css" >
< /head >

Meta Tag:

The < meta > tag provides metadata about the HTML document, such as character encoding, author, keywords, site description, or viewport settings.It is placed inside the < head > tag and helps browsers and search engines understand the page better.
Example:
< meta charset="UTF-8" >
< meta name="viewport" content="width=device-width, initial-scale=1.0" >

Title Tag:

The < Title > tag is used to set the title of a web page. It is not only visible to users in the browser tab but is also used by browsers and search engines to understand the page content, which helps improve SEO.
Example: < title > "Your page title here" < title >

Link Tag:

The < link > tag is used to connect external resources to an HTML document, most commonly for linking CSS stylesheets. It is placed inside the < head > and does not display anything on the page itself.
Example: < link rel="stylesheet" href="style.css" >

Body Tag:

The < body > tag defines the main visible content of a web page, including text, images, buttons, and more. Everything inside the < body > is rendered by the browser for the user to see and interact with.
Example:
< body >
< h1 > Welcome < /h1 >
< p > This is my website.< /p >
< /body >

Heading Tag:

HTML provides six heading tags from < h1 > to < h6 >, used to define headings from most important (< h1 >) to least (< h6 >). They help structure content and improve accessibility and SEO.
Example:
< h1 > Main Heading < /h1 >
< h2 > Subheading < /h2 >

Paragraph Tag:

The < p > tag is used to define a paragraph of text in websites. Note: "It automatically adds spacing before and after the text to separate it from other content."
Example: < p > This is a paragraph. < /p >

Anchor Tag:

The "Anchor < a >" tag is used to create hyperlinks to other web pages, files, email addresses, or sections of the same page. It uses the "href" attribute to specify the destination URL.
Example: < a href="https://example.com" > Visit Website < /a >
