Types of Tags in HTML

There are 2 types of tags in HTML

  1. Self-Closing Tag (Void Element)
  2. Paired Tag (Container Tag)

Self-Closing Tag (Void Element):

A self-closing tag does not need a closing tag because it doesn't wrap any content. It ends with /> (optional), It is used for empty elements.
Example: < br />, < image src=" " />, < hr >

Paired Tag (Container Tag):

A paired tag has an opening and a closing part and wraps around content to apply structure or styling.
Example: < p > Pargraph < /p >, < h1 > Heading < /h1 >
